`languageFn` - [language functions](https://github.com/highlightjs/highlight.js/tree/main/src/languages) to highlight your content. Function that syntax highlights the `content` (code) you provide,
brought to you by [highlight.js](https://github.com/highlightjs/highlight.js).

Some programming languages include:

- JavaScript
- TypeScript
- XML (HTML)
- CSS
- Python
- Dart (Flutter)
- Rust
- ...and so much more!

If your content is some plaintext, you can still use highlight.js' [plaintext language function](https://github.com/highlightjs/highlight.js/blob/main/src/languages/plaintext.js)
